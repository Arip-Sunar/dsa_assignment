# Documentation for Question 8  
## Sorting Algorithms with Performance Analysis



## 1. Data Structures Used

- **Array**
  - Stores randomly generated integers



## 2. Functions Implemented

### `bubbleSort()`
- Repeatedly swaps adjacent elements
- Simple but inefficient

### `selectionSort()`
- Selects smallest element and swaps

### `insertionSort()`
- Inserts elements into correct position

### `swap()`
- Swaps two values and increments swap counter



## 3. Main Function Overview

1. Takes number of elements from user
2. Generates random numbers
3. Displays unsorted list
4. Asks user to select sorting algorithm
5. Sorts the list
6. Displays sorted list
7. Shows comparisons and swaps



## 4. Sample Output

Enter number of elements: 5
Before sorting:
432 98 765 120 54

Choose sorting algorithm:

Bubble Sort

Selection Sort

Insertion Sort


After sorting:
54 98 120 432 765

Total Comparisons: 10
Total Swaps: 4



## 5. Conclusion

This program compares different sorting algorithms and shows their performance in terms of comparisons and swaps.
